# react-redux-hapi-boilerplate
React redux hapi server side rendering swagger api ui
Includes:
* React
* Redux
* Hapi.js
* server side rendering
* swagger
* React Router v4
* Webpack 2
* Sass Loader
* React Semantic Ui
* Bootstrap v4
* Sequelize.js
###### How to use for developemet:
```
git clone 
npm install
npm run dev
```
##### For production:
```
npm run production
```
##### Start only api:
```
npm start
```
